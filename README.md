## Usage
1. Click the navigation links if you want to jump down to a certain section of the page instead of scrolling. On smaller screen sizes such as tablet or mobile, you will need to click on the hamburger helper icon at the top before you can get to the navigaition links.
2. Hover over the list in each section to have a lively effect: 
    - Projects
    - Experience
    - Skills
3. Click either icon in the "Connect" section.
4. Fill out the form if you would like to leave a message and I will get back to you at my earliest convenience.
